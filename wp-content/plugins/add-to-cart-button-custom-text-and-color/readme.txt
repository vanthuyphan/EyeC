Allows to customize the Add to cart button text in WooCommerce by product type in both archive and single product pages and also allow to change woocommmerce button and font color.

== Description ==

WooCommerce is one of the most popular WordPress plugin and best solution for e-commerce. The button used to buy products has static text also same defult button and text color, so with the help of this plugin you can easily change woocommerce add to cart button text and button background and color text.


= Features: =
*   Users can easily change 'Add To Cart' button text, also there is option for Archive and Single Product page.
*   Users can easily change woocommerce button color.
*   Users can easily chnage woocommerce button text color.
*   ASCII Special Character Support
*   <a href="http://unicode-table.com/en/sets/special-symbols/" rel="nofollow" target="_blank">Unicode Character</a> Support
*   <a href="http://dev.w3.org/html5/html-author/charref" rel="nofollow" target="_blank">HTML5 Special Characters</a> Support
*   You can use available icons from your theme or plugin; like- FontAwesome, Bootstrap or material Icons
*   <a href="http://getemoji.com" rel="nofollow" target="_blank">Emoji</a> Support
*   <a href="https://wpml.org/" rel="nofollow" target="_blank">WPML</a> Compatible


<b>So you easily change woocommerce 'ADD TO CART' button text and change woocommerce background color and text color.</b>


== Plugin Settings ==

Plugin settings can be found in admin area, woocommerce -> WooCommerce Buttons


== Installation ==

Installation and uninstallation are extremely simple. You can use WordPress' automatic install or follow the manual instructions below.

= Installing: =

1. Download the package.
2. Extract it to the "plugins" folder of your WordPress directory.
3. In the Admin Panel, go to "Plugins" and activate it.
4. Go to woocommerce -> WooCommerce Buttons to use it.

= Uninstalling: =

1. In the Admin Panel, go to "Plugins" and deactivate the plugin.
2. Go to the "plugins" folder of your WordPress directory and delete the files/folder for this plugin.

= Usage: =

Using this plugin is very simple. Once it has been activated, navigate to woocommerce -> WooCommerce Buttons and chnage 'ADD TO CART' text and woocommerce button and font color.
